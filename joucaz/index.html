EasyLifeRender - Blender Addon

×
EasyLifeRender is a Blender addon that allows users to add lights and a camera around selected objects based on various presets.

Available Presets :
- Basic 3 Point Lights
- French Lights
- Tamised Lights
- Caliente Lights
- Samba Do Brazil Lights

You can use it easily by downloading and installing it in your Blender.

Blender | Python| 2024

SortHierarchy - Blender Usefull

×
SortHierarchy is a script designed to sort and organize objects within a specific scene or collection based on various criteria.

The criteria include :
- Object Type (Mesh, Light, Camera, Armature, etc.)
- Material
- Distance from the Camera

This script can be easily used by running the provided Python code in the "Scripting" section of Blender. After that, simply navigate to the "Object" section in the 3D viewport and click on "Sort Objects."

Blender | Python| 2024
